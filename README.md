-   [xordle.app](https://www.xordle.app)
-   Create rooms and play Wordle against your friends
-   Adjustable settings for:
    -   Stopping at first winnner
    -   Winning by least attempts
    -   Guess timer (Oldest word is removed from your board)
    -   Game timer (2 minute game time limit for all players)
    -   Six Attemtps (Players are only allowed 6 attempts)
    -   Chain (The word from the last game is the first word for the next game)
-   UI buit with **React** (Vite)
-   Backend built with **NodeJS** (Express)
-   Client and server state communication with **WebSocket**
